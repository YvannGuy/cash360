# Musique de Noël

Pour ajouter une musique de Noël à la page de tirage au sort :

1. Placez votre fichier audio dans ce dossier (`/public/music/`)
2. Nommez-le `christmas.mp3` (ou modifiez le nom dans le code)
3. Formats supportés : MP3, OGG, WAV

## Sources de musique de Noël libre de droits

- [Free Music Archive](https://freemusicarchive.org/)
- [Pixabay Music](https://pixabay.com/music/)
- [Freesound](https://freesound.org/)

## Note

Si aucun fichier local n'est trouvé, une musique de secours en ligne sera utilisée.
