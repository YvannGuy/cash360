# Images de conférences Masterclass Cash360

Placez ici les images des masterclass organisées dans différents pays.

## Structure recommandée

- `conference1.jpg` - Masterclass à Paris, France
- `conference2.jpg` - Masterclass à Abidjan, Côte d'Ivoire
- `conference3.jpg` - Masterclass à Dakar, Sénégal
- `conference4.jpg` - Masterclass à Douala, Cameroun
- `conference5.jpg` - Masterclass à Ouagadougou, Burkina Faso
- `conference6.jpg` - Masterclass à Bamako, Mali

## Format recommandé

- Format : JPG ou PNG
- Taille recommandée : 1200x800px minimum
- Poids : < 500KB par image pour optimiser le chargement

Les images seront automatiquement affichées dans la galerie de la page masterclass.
